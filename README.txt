# 🌟 Anime Quotes Generator

A sleek and minimal Flask-based web app that displays random anime quotes — and lets you add your own! Built with Python, HTML, CSS, and JavaScript. Features a clean, tech-inspired interface using black, gold, and white.

---

## 📸 Preview

> 🎬 **Click "Get your Quote"** to see a random anime quote, or add your favorite one using the form.

---

## 🚀 Features

- 🎴 Random Anime Quote Generator
- ➕ Add your own quotes (stored in `quotes.json`)
- 💾 Simple file-based storage (no database)
- ✨ Tech-styled UI with custom colors and smooth animations
- 🖥️ Built using:
  - Python & Flask
  - HTML, CSS, JavaScript

   ```bash

   cd Quotes_API
2. Create virtual environment

   python -m venv venv
   venv\Scripts\activate    # On Windows

3. Install dependencies

pip install -r requirements.txt

4. Run the app
